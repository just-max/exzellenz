Text Generator

Use this tool to generate text set in the EXZELLENZ typeface. Enter the desired text and select the options below. The font only contains glyphs for upper case letters (A-Z) but will render lower case characters as their upper case equivalents. The SVG format option embeds the EXZELLENZ font, but this only works when viewing in a browser. If the font is installed locally, the SVG should work in any viewer. The font can be downloaded from the GitHub page.

The EXZELLENZ font is licensed under the SIL Open Font License, Version 1.1.
